The toolbar component's data is set via a service. Therefore, each toolbar on this site has the same data.

Title handling

You are able to set the title of the toolbar via: this.toolbarService.toolbarTitle = "Toolbar Demo Title"; If no title is specified, the website title (e.g. on this site it's Toolbar Component) will be set.

Additionally, you can set a sub title via: this.toolbarService.setDataTitle("Sub title");

Toolbar actions

Three actions are available in the toolbar:

  • Main Action
  • Toolbar Action
  • Back Action

Back Action

You can specify a back action via: this.toolbarService.addBackAction('/home'); This will render a back arrow in front of the toolbar title, which navigates to the specified route.

It's also possible to define an absolute route to a different website by setting the 'isAbsoluteUrl' flag true: this.toolbarService.addBackAction('https://www.example.com', true); The same back arrow as above will appear, this time using a [href] instead of a [routerLink].
